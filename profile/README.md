# üíª Installation Process

> 

```
git clone --recursive <https://github.com/GeubDDong/Frontend.git>
git clone --recursive <https://github.com/GeubDDong/Backend.git>
```

<br>

> <b>Set .env in the backend folder </b>
> 

```
PORT=

DB_TYPE=
DB_HOST=
DB_PORT=
DB_USERNAME=
DB_PASSWORD=
DB_DATABASE=
DB_SYNCHRONIZE=
DB_LOGGING=

REDIS_HOST=
REDIS_PORT=
REDIS_PASSWORD=

KAKAO_HOST_NAME = 
KAKAO_REST_API_KEY = 
KAKAO_SECRET_KEY = 
KAKAO_REDIRECT_URI = 

JWT_SECRET = 
JWT_EXPIRE_IN = 
REFRESH_JWT_SECRET = 
REFRESH_JWT_EXPIRE_IN =
```
